## 🚀 About

Gloss Pharma ERP is a full-stack Enterprise Resource Planning system designed for pharmaceutical businesses.

It streamlines daily operations by managing inventory, employees, purchases, sales, reports, AI assistance, document management, and secure authentication—all from a single dashboard.

# 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/yourusername/Gloss-Pharma-ERP.git
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
